Introduction
============

Code emamples for "Hardened Plone" Penn State Plone Symposium Presentation.

Running the buildout
--------------------

Just the normal thing::

    git clone git://github.com/vangheem/psusymp.hardenedplone.git
    cd psusymp.hardenedplone
    python2.6 bootstrap.py
    ./bin/buildout
